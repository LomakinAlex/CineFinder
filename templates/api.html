Do you want to embed into the program
an easy way to find a movie and find movies based on the plot?
You can use our API!
We use the API for our service to easily and quickly work with finding movies based on the plot.

In order to use it you need to use the URL:

127.0.0.1:5000/api/ask?plot=New%20York&amount=50

plot: Parameter with the plot

amount: Parameter, that returns the maximum number of movies found


Python
import requests plot = "New York" amount = 5 req = requests.get("127.0.0.1:5000/api/ask", params={"plot":plot,"amount":amount}) content = req.content print(content) >>> [ >>> "Mission to Moscow", >>> "Zoolander 2", >>> "Into the Blue", >>> "Anything Goes", >>> "Flirting" >>> ]
